Responsabilidades

Legibilidade

Legibilidade é um ponto muito importante para todos os elementos que buscamos alcançar. É preciso tornar seu software legivel para seus clientes, e é preciso tornar seu software legivel para o proximo que trabalhar nele.

Na construção de um software web, tudo é muito delicado, por tanto é preciso ser bastante cuidadoso com todas as estruturas.

Por isso tenho total responsabilidade em buscar ser o mais apreensivo com seu produto, para que não venha a ter problemas futuramente.

Responsividade

No mundo todo, a diversidade dos dispositivos acaba se tornando uma questão pessoal, sendo assim, cada sistema deve empregar uma responsabilidade de transmitir a sua mensagem para um cliente da melhor forma possivel.

Por tanto, seu site precisa se adaptar a esses diferentes dispositivos, com diferentes tamanhos de tela.

Com tudo tenho o dever de produzir um sistema que possa chegar a qualquer um de seu publico alvo.

Flexibilidade

Todos os modulos, métodos e classes podem ser independentes e reutilizaveis, dando maior flexibilidade para o produto, tornando outros projetos com uma medida de evolução superior a tal projeto.

Isso faz com que seu produto evolua em uma velocidade absurda.
